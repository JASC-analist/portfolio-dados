# Análise de Cancelamento de Clientes

# Descrição
Projeto desenvolvido em Python com o objetivo de analisar dados de cancelamento de clientes, realizando tratamento dos dados e criação de visualizações para identificar padrões.

# Ferramentas Utilizadas
- Python
- Pandas
- Plotly

# O que foi feito
- Importação e visualização dos dados
- Tratamento de dados (remoção de valores nulos e inconsistentes)
- Análise da taxa de cancelamento
- Criação de gráficos para análise visual

# Resultado
Identificação de padrões relacionados ao cancelamento de clientes, auxiliando na análise e tomada de decisão baseada em dados.

# Como executar o projeto
1. Instale o Python que preferir e selecione ele em "Select Kernel" no canto superior direito no [AnalisandoDados.ipynb]
2. Instale as bibliotecas necessárias: pip install pandas plotly
